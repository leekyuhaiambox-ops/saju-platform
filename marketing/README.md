# 사주명리 플랫폼 — 마케팅 콘텐츠 패키지

복사·붙여넣기로 바로 쓸 수 있는 SNS 콘텐츠 자료입니다.

## 파일 안내

| 파일 | 용도 |
|---|---|
| `instagram-posts.md` | 인스타그램 피드/릴스 게시물 30개 (1달치) |
| `twitter-posts.md` | X(트위터)·스레드 게시물 30개 |
| `kakao-channel.md` | 카카오톡 채널·오픈채팅 운영 자료 |
| `naver-blog-posts.md` | 네이버 블로그·티스토리용 SEO 친화 글 5편 |
| `youtube-shorts.md` | 유튜브 쇼츠·인스타 릴스 30초 영상 대본 10편 |
| `hashtags.md` | 플랫폼별 추천 해시태그 모음 |
| `content-calendar.md` | 4주치 일간 게시 캘린더 |
| `seo-strategy.md` | SEO 키워드·내부 링크 전략서 |
| `bio-templates.md` | SNS 프로필 소개·링크 모음 |
| `community-strategy.md` | 커뮤니티(에브리타임·디시·네이트판) 홍보 전략 |

## 운영 원칙

1. **자랑이 아닌 가치** — "이 사이트 좋아요"가 아니라 "이 일주 사람 특징이 이렇대" 처럼 정보 중심으로
2. **하루 1~2개** — 너무 많이 올리면 알고리즘 페널티. 꾸준함이 핵심
3. **링크는 마지막에** — 본문에 가치 있는 내용 먼저, 링크는 댓글이나 마지막에
4. **답글·DM 활성화** — 사주 풀이 댓글에 답 달아주면 팔로워 빠르게 늘어남
5. **금기어 피하기** — "100%", "확실히", "운명적으로" 같은 단정 표현은 광고 정책 위반 우려

## 첫 2주 추천 진행

- 1주차: 인스타·X 각 하루 1개씩 + 네이버 블로그 1편
- 2주차: 카카오 오픈채팅 개설 + 유튜브 쇼츠 첫 영상
- 트래픽이 일 100명 넘어가면 → AdSense 신청 + Meta Pixel로 광고 캠페인 검토
