## 게시 시간 (한국 기준)

| 채널 | 권장 시간 | 이유 |
|---|---|---|
| 인스타그램 | 매일 21:00 ~ 22:00 | 한국 사용자 활동 피크 |
| X (트위터) | 매일 21:30, 영문은 미국 동부 09:00 | 두 시장 동시 |
| 네이버 블로그 | 주 1회 일요일 오후 | 검색 노출 누적 |
| Medium | 주 1회 일요일 미국 시간 | 영어권 |

## 자동 발행 가능한 것

- **티스토리**: 카카오 REST API 키로 자동 게시 가능 (별도 봇 작성)
- **Medium**: Integration Token (구버전 발급 가능자만)
- **나머지(인스타·X·네이버)**: 모두 수동 — 위험성 때문에 자동 게시 비추천

## 효율 팁

- 한 주분 콘텐츠를 일요일에 미리 준비 → 매일 5분만 투자
- 인스타·X는 같은 콘셉트로 변형 (인스타는 캡션 길게, X는 짧게)
- 30일 완주 후 → 인스타 인사이트로 어떤 게시물 반응 좋았는지 분석 → 다음 달 비슷한 톤으로
