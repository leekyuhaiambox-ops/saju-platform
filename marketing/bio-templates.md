## SNS 일관성 체크리스트

모든 SNS에서 다음 4가지를 통일해야 사용자가 "같은 브랜드"로 인식합니다.

- [ ] 프로필 이미지 (모든 곳 동일)
- [ ] 배경 이미지 / 커버 (가능한 곳)
- [ ] 채널명 / 아이디 (`tarofortune` 또는 `사주명리`)
- [ ] 소개글 (핵심 메시지 일관)
- [ ] 링크 (`tarofortune.pythonanywhere.com` 또는 Linktree)
