# 커뮤니티 홍보 전략 (디시·에브리타임·네이트판·카카오 오픈채팅)

광고 티 나는 글은 즉시 차단·삭제됩니다. **사용자처럼 자연스럽게 정보 공유**가 핵심.

---

## 1. 카카오 오픈채팅

가장 효율 높은 채널. 한국 사용자 즉시 도달.

### 채팅방 개설

**이름**: `사주 일주 풀이 공유방 (무료)` (검색 노출 핵심 키워드)

**소개**:
```
60갑자 일주 풀이를 무료로 받고 공유하는 방입니다.
- 본인 일주 닉네임 ([갑자] 닉네임 형식)
- 풀이 결과 공유 환영
- 광고·홍보 글 금지

운영자 도구: 무료 사주 사이트 (방 공지에 링크)
```

**검색 키워드 (5개)**:
```
사주, 일주, 사주풀이, 무료사주, 명리학
```

**입장 질문 (필터링용)**:
```
1. 본인 닉네임 + 본인 일주
2. 사주 풀이를 처음 받아보시나요?
```

### 운영 원칙

- **공지에만 사이트 링크 1번**. 본문에 반복 게시 X.
- 주 1회 "오늘의 일진 ○○" 게시 (재방문 유도)
- 멤버 일주 풀이 요청 시 → 사이트 링크 안내
- 매주 인기 일주 통계 공유 ("우리 방 갑자 일주가 가장 많네요")

### 멤버 100명 도달 후

- 운영진 2~3명 모집
- 매주 토요일 사주 Q&A 시간
- 멤버 후기 이벤트 (사이트 사용 후기 1개 → 추첨 1명 운세 풀이)

## 2. 디시인사이드

### 추천 갤러리

- **사주갤러리** (sajugall) — 가장 활성, 메인 타겟
- **점성술갤러리** — 보조 타겟
- **미스터리갤러리** — 폭넓은 호기심층
- **여시갤러리** (여성 사용자 많음)

### 게시 전략 (광고 X, 자연스러운 후기)

**제목 예시:**
```
[일주 풀이] 갑자 일주인데 진짜 학자형 맞나..?
사주 공부 시작했는데 일주 풀이 보면 신기함
○○○ 풀이 받았는데 이거 진짜 정확한가
입춘 전에 태어난 사람들 띠가 다르다는데 이거 정상?
```

### 글 작성 팁

- **본인 후기** 형식. "나는 ○○ 일주인데 이게 진짜 맞는 것 같다" 식.
- 본문에 사이트 링크 1회. (광고처럼 보이지 않게 "내가 본 사이트는 ○○인데..." 식)
- 답글에 적극 응대. 디시는 답글 활성도가 곧 게시물 노출.
- **새벽~오전 게시 X**. 점심·저녁이 골든타임.

### 절대 금지

- "이 사이트 추천합니다" 같은 명시적 홍보 (즉시 차단)
- 본인이 만든 사이트라고 자랑 (반발 심함)
- 댓글 자작자 (들통나면 매장)

## 3. 에브리타임 (대학생)

### 게시판 선택

- **익명게시판** — 가장 활성
- **자유게시판** — 학교별 풍토 다름
- **운세 카테고리** (있는 학교)

### 게시 전략

**제목:**
```
사주 처음 봤는데 신기해서 공유 (오늘 본 결과)
○○ 일주가 ENFP 많다는데 진짜?
연인이랑 사주 궁합 봤는데 결과 충격
취업 사주 봤는데 ○○ 분야가 적성이래
```

### 글 작성

- 캡처 1~2장 첨부 (본인 결과 페이지)
- 본문에 사이트 URL 명시 (광고 X 후기 형식)
- "혹시 본인 결과 어떤지 댓글로 공유해주세요" → 댓글 트래픽

### 시즌 활용

- 3월·9월 개강 시즌 → 신학기 운세
- 11월 수능 시즌 → 수능 사주
- 12월 연말 → 신년 운세

## 4. 네이트판

### 추천 카테고리

- **사주/운세** — 가장 직접적
- **결혼/궁합** — 궁합 컨텐츠
- **연애** — 연애 사주
- **고민/사연** — Q&A 형식

### 글 작성 팁

- 네이트판은 **사연 형식** 글이 잘 읽힘.
- "남친이랑 사주 궁합 봤는데 진짜 충격..."
- "엄마가 자꾸 ○○ 일주 만나라는데 이게 뭐 다른가요?"
- 본문 끝에 "참고로 제가 본 사이트는 ○○ 입니다" 자연스럽게

## 5. 네이버 카페

### 추천 카페

- **사주명리학 카페** (검색 시 회원 1만+ 카페 4~5개)
- **결혼 준비 카페** — 궁합 관련
- **연애 카페** — 궁합·연애운
- **수험생/취준생 카페** — 진로 사주

### 가입 후 활동 패턴

1. 첫 2주: 댓글만 활발하게 (등급 올리기)
2. 3주차: 본인 사주 후기 + 사이트 언급 1회
3. 이후: 매주 1회 정보성 글 (사이트 자연스럽게 노출)

**팁:** 카페마다 '광고 글 기준'이 다르니 운영 규칙 먼저 정독.

## 6. 인스타그램 DM 마케팅 (조심스럽게)

### 대상

- 본인이 팔로우하는 사주 관련 계정 팔로워
- 본인 게시물에 좋아요 누른 사람들

### DM 내용 (절대 광고 X)

❌ "저희 사이트 한번 봐주세요" (스팸)

✅ "○○님 팔로우 감사합니다. 혹시 본인 일주가 뭐예요? 풀이 무료로 알려드릴게요."

→ 답장 오면 → "○○ 일주시면 이런 특성이에요" + 사이트 링크 자연스럽게

**일 DM 한도**: 인스타 정책상 일 50건 미만. 넘으면 계정 임시 정지.

## 7. 유튜브 댓글 마케팅

### 대상

- 큰 사주 유튜버 영상 (조회수 10만+)
- 댓글 상위 노출 노리기

### 댓글 작성

- 영상에 대한 진심 어린 코멘트 + 본인 일주 언급
- "저는 ○○ 일주인데 영상에서 말씀하신 거 너무 맞아요"
- 사이트 URL은 직접 쓰지 말 것 (영상 광고 정책 위반)

### 댓글 시간

- 영상 업로드 30분~2시간 이내 댓글이 상단 노출 확률 높음
- 알림 설정해서 빠르게 댓글
