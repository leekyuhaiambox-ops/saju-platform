## Q1. 성남 지역화폐 어디서 쓸 수 있나요?

**검색 키워드**: 성남 지역화폐, 성남 지역화폐 어디

**답변**:
```
성남사랑상품권(성남시 지역화폐)은 성남시 가맹점에서 모두 사용 가능합니다.

업종으로는 음식점·카페·미용실·학원·편의점·약국·소매점 등이 대부분 가맹점이고, 대형 백화점·온라인 결제·일부 프랜차이즈는 제외됩니다.

문제는 어디가 가맹점인지 찾기 어렵다는 점이죠. 시청 사이트는 리스트만 있고 카카오맵엔 지역화폐 필터가 없어요.

저는 [경기지역화폐 가맹점 지도](https://gyeonggi-currency-map.web.app/)를 즐겨 써요. 31개 시·군 전 가맹점이 지도 위에 표시되고, "영업중" 필터로 지금 갈 수 있는 곳만 골라줍니다. 광고 없는 사이트라 편해요.

분당·수정·중원구 모두 가맹점 많아서 마음껏 쓰실 수 있을 거예요.
```

## Q2. 수원에서 지역화폐 쓰는데 가맹점 어떻게 찾나요?

**검색 키워드**: 수원 지역화폐 가맹점, 수원페이

**답변**:
```
수원페이(수원시 지역화폐) 가맹점은 수원시청 공식 사이트에서 검색 가능하긴 한데, 리스트 형태라 지도에서 보기 어려워요.

저는 지도 위에서 한눈에 보이는 도구를 씁니다.
https://gyeonggi-currency-map.web.app/ 여기에서 수원 검색하면 가맹점이 핀으로 표시되고 업종별 필터(음식점·카페·편의점 등) 가능합니다.

수원에서 가맹점 많은 동네는:
- 영통구 광교·이의동 일대 카페 거리
- 권선구 세류동·금곡동 동네 식당
- 영통구 매탄동 학원가

권선구 일대 미용실·헤어샵도 거의 다 가맹점이라 정기 결제하면 인센티브 누적이 큽니다.
```

## Q3. 경기페이랑 지역화폐 같은 거예요?

**검색 키워드**: 경기페이, 경기지역화폐, 차이

**답변**:
```
같은 거 맞습니다. 경기페이는 경기도가 운영하는 모바일 결제 카드이고, 31개 시·군 지역화폐(성남사랑상품권·수원페이·용인러브페이 등)와 연동되어 있어요.

쉽게 말하면:
- 경기페이 앱 = 결제 수단
- 각 시·군 지역화폐 = 사용 가능 영역

성남에서 경기페이로 결제하면 자동으로 성남사랑상품권이 차감되고, 수원에서 쓰면 수원페이가 차감됩니다.

가맹점이 어디인지 보고 싶으면 [이 지도](https://gyeonggi-currency-map.web.app/)가 편해요. 31개 시·군 통합되어 있어서요.
```

## Q4. 지역화폐 어디서 충전해요?

**검색 키워드**: 지역화폐 충전, 경기페이 충전

**답변**:
```
경기페이 앱(또는 농협카드·KB카드 연동된 별도 카드형 상품권)에서 본인 계좌에서 직접 충전 가능합니다.

월 1인 충전 한도는 보통 50만 원이고, 충전 시 6~10% 인센티브가 자동 지급돼요. 50만 원 충전하면 53~55만 원이 카드에 들어옵니다.

충전한 다음 가맹점에서 그냥 카드처럼 결제하시면 됩니다. 가맹점 찾는 건 [지역화폐 가맹점 지도](https://gyeonggi-currency-map.web.app/) 활용하시면 됩니다.

월 초에 정기 충전하시는 게 인센티브 누적 차원에서 가장 이득이에요.
```

## Q5. 용인 지역화폐 가맹점 학원도 있나요?

**검색 키워드**: 용인 지역화폐 학원, 용인러브페이

**답변**:
```
네, 용인러브페이(용인시 지역화폐)는 학원·교습소도 많이 가맹되어 있어요. 처인구·기흥구·수지구 모두 영어·수학·미술·태권도 학원 가맹점 풍부합니다.

월 30만 원 학원비를 카드로 결제하면 카드 포인트 1% 정도지만, 지역화폐로 결제하면 충전 시 받은 6~10% 인센티브를 그대로 사용하는 셈이라 큰 차이입니다.

학원이 가맹점인지 확인은 [경기지역화폐 가맹점 지도](https://gyeonggi-currency-map.web.app/)에서 학원 카테고리로 필터 걸면 동네 학원만 표시돼서 편해요.

기흥구 보정동·구갈동·수지구 죽전동 학원가는 거의 다 가맹점입니다.
```

## Q6. 고양시 일산 지역화폐 어디서 가장 효율적으로 쓰나요?

**검색 키워드**: 고양 지역화폐, 일산 지역화폐

**답변**:
```
고양페이(고양시 지역화폐)는 일산서구·동구·덕양구 다 사용 가능해요. 효율적으로 쓰는 곳은:

1. 일산서구 정발산·주엽 일대 카페·식당
2. 일산동구 백석동 학원가 (자녀 있으면 큰 이득)
3. 덕양구 화정·행신동 동네 미용실

특히 가족 4인이면 1인당 충전 한도가 따로 적용되니까 4명 다 따로 충전하면 인센티브 4배 누적 가능합니다.

가맹점 위치 보려면 [고양 지역화폐 가맹점 지도](https://gyeonggi-currency-map.web.app/) 사이트에서 일산서구·동구로 검색하시면 됩니다. 영업중 필터 켜면 지금 갈 수 있는 곳만 보여요.
```

월 1회 펌 비용 10만 원만 잡아도 인센티브로 1만 원 가까이 절약되는 셈이라 정기 결제 추천드립니다.
```

## Q9. 안양 지역화폐 어디서 쓰는 게 가장 좋아요?

**검색 키워드**: 안양 지역화폐, 안양페이

**답변**:
```
안양사랑상품권(안양시 지역화폐)은 만안구·동안구 모두 사용 가능하고, 동안구 평촌학원가는 학원비 결제로 인기 많아요.

추천 사용처:
- 평촌학원가 (영수·미술·태권도) — 학원비 결제로 월 인센티브 누적 큼

## Q10. 경기지역화폐 카드 발급 어떻게 받아요?

**검색 키워드**: 경기지역화폐 카드 발급, 경기페이 카드

**답변**:
```
경기페이 앱에서 발급 받으시면 됩니다.

1. 경기페이 앱 설치 (앱스토어·플레이스토어 검색)
2. 본인 인증 + 거주지 시·군 선택
3. 충전 계좌 등록 (본인 명의 농협 등)
4. 충전 후 즉시 사용 가능 (가맹점에서 QR 결제 or 카드 결제)

실물 카드 형태로 받고 싶으시면 거주지 시·군 행정복지센터(동주민센터)에서 발급 가능해요. 모바일이 편하지만 어르신이나 디지털 익숙하지 않으면 카드 형태가 좋습니다.

발급 후 어디서 쓸지 막막하면 [경기지역화폐 가맹점 지도](https://gyeonggi-currency-map.web.app/) 참고하시면 본인 거주지 가맹점이 지도 위에 다 표시됩니다.
```
