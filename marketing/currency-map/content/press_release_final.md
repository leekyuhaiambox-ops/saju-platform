### [버전 B — 긴 보도자료 (800자, 종합지·전국지)]

**제목**: 31개 시·군 경기지역화폐 가맹점 통합 지도 등장 — 민간 개발자, 무료 PWA로 공공 데이터 활용

```
[수원=종합] 경기도 1,400만 거주민이 사용하는 경기지역화폐(경기페이) 가맹점을 31개 시·군 통합 지도에서 검색할 수 있는 무료 웹 서비스가 정식 공개됐다.

'경기지역화폐 가맹점 지도'(https://gyeonggi-currency-map.web.app)는 민간 개발자 이 모씨가 약 한 달간 개발한 Progressive Web App(PWA)로, 별도 앱스토어 설치 없이 휴대폰 브라우저에서 바로 사용 가능하며 홈 화면 추가 시 네이티브 앱처럼 작동한다.

기존 경기페이 공식 앱과 각 시·군별 공식 사이트는 가맹점 검색을 리스트 형태로 제공해 왔다. 이로 인해 사용자가 지역화폐를 충전한 후에도 "어디서 쓸지 모르겠다"는 불편이 빈번했다.

새 서비스는 다음 기능으로 이 문제를 해결한다:
- 수원·성남·용인·고양 등 31개 시·군 통합 검색
- 음식점·카페·편의점·미용실·학원 등 업종별 필터링
- 현재 시각 기준 '영업중' 가맹점만 표시하는 실시간 필터
- 가맹점 정보를 카카오톡으로 1초 공유
- 웹앱(PWA)으로 앱 설치 없이 즉시 사용

가맹점 데이터는 경기도 공공 데이터 포털의 공식 데이터를 활용하며, 매주 자동 갱신된다. 서비스 운영비는 Google AdSense와 Kakao AdFit 광고로 충당하며 사용자 결제·구독은 받지 않는다.

기술 구현은 React + Vite + Leaflet 조합으로 Firebase Hosting에 배포되어 있다. 서버리스 아키텍처라 트래픽 증가에도 운영비 부담이 적다.

개발자 이씨는 "지역화폐 충전 후 안 쓰면 결국 인센티브 5~10%를 받지 못하는 셈인데, 가맹점 검색이 너무 어려워서 시작하게 됐다"며 "장기적으로는 31개 시·군 행정복지센터·지자체와 협력해 가맹점 데이터의 실시간성을 더 높이고 싶다"고 밝혔다.

서비스는 https://gyeonggi-currency-map.web.app 에서 무료 이용 가능하다.

문의: leekyuha.iambox@gmail.com
```
