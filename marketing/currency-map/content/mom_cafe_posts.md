# 맘카페 게시 카피 — 경기지역화폐 (31개 시·군용)

> **운영 원칙**:
> - 각 카페 가입 → 등업(댓글 5~10개) → 자유게시판 1회 게시
> - **절대 금지**: 같은 카피 31개 카페 일괄 게시 → 즉시 차단
> - 1주 1게시, 시·군 이름·상권만 바꿔서 변형 게시
> - 광고로 분류되지 않게 "공유" / "꿀팁" 느낌으로

---

자녀 학원비 결제도 가맹점 많아서 좋아요. 분당구 정자·서현·미금 학원가는 거의 다 가맹점이더라고요.

### 톤 B — 자영업자 친화형

**제목**: [수원맘] 우리 동네 음식점 살리는 꿀팁 (경기페이)

```
수원에서 식당 운영하시는 분 동네에 많죠.

저도 자영업자 입장에서 단골이 지역화폐 가맹점 찾기 어려운 게 늘 답답했어요. 손님은 "여기 가맹점이에요?" 물어보고 모르겠다 답하면 그냥 카드 결제하고 가버리는데, 사실 우린 가맹점인데도요.

최근에 [경기지역화폐 가맹점 지도](https://gyeonggi-currency-map.web.app/) 사이트가 나왔는데, 31개 시·군 가맹점이 자동으로 다 표시되더라고요. 우리 가게도 가맹 등록만 되어 있으면 자동 노출이라 좋았어요.

손님분들도 식당 미리 찾고 오시고, 자영업자 입장에선 가맹점이라는 게 자연스럽게 드러나는 셈이에요.

수원 영통구·권선구·팔달구·장안구 다 같은 사이트에서 검색 가능하니까 동네 식당 단골 만드시는 데 활용해 보세요.

광고 아니고요 자영업자 입장에서 알게 된 도구라 공유합니다.
```

### 톤 C — 학원·교육 관심형

**제목**: [용인맘] 자녀 학원비 지역화폐로 결제하면 진짜 이득

```
초등·중등 자녀 키우시는 분께 작은 꿀팁 공유드려요.

자녀 학원비가 월 30~50만 원 큰 지출인데, 카드로 결제하면 카드 포인트가 0.5~1% 정도예요. 그런데 용인러브페이(용인시 지역화폐)로 결제하면 충전 시 받은 6~10% 인센티브를 그대로 학원비에 쓰는 셈이에요.

기흥구 보정동·구갈동·수지구 죽전동 학원가는 영어·수학·미술·태권도 학원 거의 다 가맹점이에요. 가맹 여부 확인은 [경기지역화폐 가맹점 지도](https://gyeonggi-currency-map.web.app/)에서 학원 필터로 보면 빨라요.

저희 집은 자녀 2명 학원비 합쳐 월 60만 원인데 지역화폐로 돌리면 연간 인센티브만 40만 원 정도 쌓여요. 이걸로 가족 외식하면 한 달에 한 번 무료 외식하는 셈이죠.

처인구·기흥구·수지구 다 같은 방식 가능하니까 자녀 학원비 결제 한번 점검해 보세요.
```

## 게시 후 알고리즘 활용

1. **즉시 댓글 활동** — 같은 카페에서 다른 글 5개 정도 댓글 → 본인 글 알고리즘 추천 점수 ↑
2. **24시간 응답** — 본인 글에 댓글 달리면 24시간 내 답글 (대화율 = 추천 지표)
3. **1주일 후 후속 글** — 같은 카페에서 관련 정보 글 1개 더 (시리즈로 보이게)

---

## 등업 가이드 (각 카페 공통)

대부분의 맘카페 등업 조건:
- 가입 7일 이상
- 댓글 5~10개 작성
- 출석체크 3일 이상
- 자기소개 작성

**효율적 등업 패턴**: 가입 직후 → 자기소개 → 신입 인사 게시판에 댓글 → 3일 출석체크 → 정보 공유 게시판에 도구 게시.

---

## 절대 금지 사항

❌ **같은 카피 31개 카페에 동시 게시** → 즉시 차단 + 운영자 차단
❌ **광고 의도 명백한 표현** ("저희 사이트", "구매 링크" 등)
❌ **카페 운영자 별도 협의 없이 광고성 게시** → 신고 사유
❌ **회원 등업 안 된 상태에서 게시** → 자동 삭제·차단
