## 들어가며

경기도 1,400만 거주민이 사용 가능한 **경기지역화폐**(경기페이)는 충전 시 6~10% 인센티브가 자동 지급되는 모바일 카드형 결제 수단이다. 31개 시·군마다 명칭이 약간씩 다르지만(성남사랑상품권·수원페이·용인러브페이 등), 공통적으로 가맹점에서 결제하면 즉시 차감되는 구조다.

문제는 가맹점 검색 인터페이스. 각 시·군 공식 사이트는 따로 분산되어 있고, 카카오맵·네이버지도에는 "지역화폐" 카테고리 필터가 없다. 본 글에서는 이 문제를 해결하는 무료 PWA **경기지역화폐 가맹점 지도**(https://gyeonggi-currency-map.web.app/)를 리뷰한다.

## 핵심 결론

| 항목 | 평가 |
|---|---|
| 통합 범위 | 31개 시·군 단일 인터페이스 |
| 검색 UX | 업종 필터 + 영업중 토글 + 카카오톡 공유 |
| 기술 스택 | React + Vite + Leaflet, Firebase Hosting, PWA |
| 비용 | 무료 (Google AdSense + Kakao AdFit 노출) |
| 적합 사용자 | 경기도민 1,400만 + 자영업자 + 외부 방문객 |

## 기능 상세 리뷰

### 1. 통합 31개 시·군 검색

기존 지역화폐 가맹점 검색은 **각 시·군 사이트마다 별도 인터페이스**를 사용해야 했다. 성남 거주자가 수원 출장에서 가맹점을 찾으려면 수원시 사이트를 따로 열어야 했다.

본 사이트는 이를 단일 지도로 통합했다. 좌측 상단 검색창에 도시명·주소·읍면동을 입력하면 해당 지역으로 지도가 이동하고 가맹점 핀이 표시된다.

### 2. 업종별 칩 필터

화면 상단의 칩 그룹으로 업종 필터링이 가능하다:
- 음식점
- 카페
- 편의점
- 학원
- 미용실
- 의료기관
- 소매·잡화

복수 선택이 가능하며, 즉시 지도에 반영된다.

### 3. 영업중 실시간 필터

각 가맹점의 영업시간 데이터를 활용해 **현재 시각 기준 영업 중인 곳만 표시**하는 토글이 있다. 헛걸음을 막아주는 기능으로, 평일 저녁이나 주말 야간 검색 시 특히 유용하다.

### 4. PWA 설치

사이트는 Progressive Web App(PWA)로 구현되어 있어 별도 앱 설치 없이도:
- iOS Safari: "홈 화면에 추가" 메뉴
- Android Chrome: "앱 설치" 자동 배너
- 데스크탑 Chrome: 주소창 우측 설치 아이콘

으로 네이티브 앱처럼 실행 가능하다. Service Worker 캐싱으로 오프라인 일부 작동도 지원한다.

### 5. 카카오톡 공유 SDK 통합

가맹점 핀 클릭 시 표시되는 카드에 **카카오톡 공유 버튼**이 있다. 클릭하면 친구·단톡방에 가맹점 정보가 OG 카드 미리보기와 함께 전송된다. 한국 사용자에게는 거의 필수 기능.

## 사용 시나리오 — 페르소나별

### A. 일반 경기도민 (소비자)

50만 원 충전 시 인센티브 5~10%로 지급되는 카드를 어디서 쓸지 검색. **월 초 충전 → 학원비·미용실비·외식비 등 큰 지출을 가맹점에서 우선 결제** 전략이 효율적이다.

### B. 자영업자·가맹점주

본인 가게가 지도에 노출되는지 확인. 미노출 시 시청 일자리경제과에 가맹 등록 신청. **31개 시·군 통합 노출**이라 다른 도시 출장객도 가게를 찾을 수 있다.

### C. 외부 방문객

수원 화성·남양주 양수리·고양 일산 호수공원 등 관광지 방문 시 식당·카페 검색. **앱 설치 없이 PWA 사용 가능**해 일회성 사용에 적합.

## 개선 여지 (관찰 사항)

1. **Sitemap.xml에 루트 URL 1개만** — 31개 시·군 deep URL이 색인되지 않음. Vite SSG 도입 시 시·군별 정적 페이지 생성 가능.
2. **검색엔진 사이트 인증 미적용** — Google Search Console, Naver 서치어드바이저 등록 시 검색 노출 가속.
3. **PWA 설치 안내 부족** — 첫 방문자가 PWA 설치를 알기 어려움.

## 결론

**기존 시·군별 분산 가맹점 검색의 통합 솔루션**으로 충실히 기능한다. UX는 카카오맵 수준으로 다듬어져 있으며, "영업중" 실시간 필터는 다른 어떤 도구도 제공하지 않는 차별점.

기술적 관점에서도 React + Vite + Leaflet + Firebase Hosting의 모범적 PWA 사례. 한국 지역 서비스에서 PWA가 잘 작동하는 드문 예시다.

경기도 거주자라면 즐겨찾기·홈 화면 추가를 권한다. 1년에 5만 원 인센티브를 받으려면 이런 도구가 결국 필요하다.
