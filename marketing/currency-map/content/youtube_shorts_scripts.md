## #1 — "성남 지역화폐 카페 BEST 5" (45초)

**훅(0~3초)**: "성남 지역화폐 어디서 쓰지 고민이라면"

**시나리오**:
```
[0~3초] 자막: "성남 지역화폐 어디서 쓰지?"
화면: 검은 배경 + 흰 자막

[3~8초] 자막: "이 지도, 한 번 보세요"
화면: 사이트 진입, 지도 + 핀들 표시

[8~15초] 자막: "성남 검색 → 카페 필터 → 영업중 토글"
화면: 검색창에 "성남" 입력 → 카페 칩 클릭 → 영업중 토글 ON
(화면 녹화 가속 1.5x)

[15~30초] 자막: BEST 5 카페 노출
- 정자동 카페 ○○
- 수내동 카페 ○○
- 야탑동 카페 ○○
- 서현동 카페 ○○
- 분당동 카페 ○○
화면: 핀 차례로 클릭 → 카페 정보 카드 줌인

[30~40초] 자막: "주말 1만원 카페 = 1천원 인센티브"
화면: 결제 화면 캡처 또는 영수증 짧게

[40~45초] 자막: "검색: 경기지역화폐 가맹점 지도"
화면: 사이트 메인 화면 + URL
```

**해시태그**: `#경기지역화폐 #성남카페 #분당맛집 #지역화폐사용처 #shorts`
