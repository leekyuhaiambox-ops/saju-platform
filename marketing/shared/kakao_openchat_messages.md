> **운영 원칙**:
> - 가입 → 30일 일반 활동 (질문·답변·정보 공유) → 도구 소개 1회

## tarofortune (사주명리) — 오픈채팅 5종

**검색 키워드**: "사주", "운세", "타로", "명리", "MBTI 사주"

### 메시지 #1 — 신년 운세 시즌
```
새해라 사주 무료 풀이 사이트 있어서 공유드려요.
tarofortune.pythonanywhere.com — 일주·월주·년주·시주 자동 계산 + 60갑자 풀이까지 한국어로 풀어줘서 처음 보는 분도 쉽게 이해 가능.

광고 없는 사이트라 부담 없이 보세요. 일진(오늘의 운세)도 매일 자동 갱신됩니다.
```

### 메시지 #2 — 일상 운세 톤
```
오늘 일진 어떨까 궁금하실 때 빠르게 확인 가능한 사이트가 있어요.
tarofortune.pythonanywhere.com → 메인에서 '오늘의 일진' 클릭하면 5초 안에 결과.

생년월일·시 입력 안 해도 그 날의 갑자만 보고 일진 풀이 보는 거라 가볍게 보기 좋아요.
```

### 메시지 #3 — 자기이해 톤
```
"MBTI는 알겠는데 사주는 잘 모르겠어요" 하시는 분 많을 듯.
tarofortune.pythonanywhere.com 에서 일주(태어난 날의 갑자) 한 번 보세요. 60갑자별로 성향·강약·연애·직업 풀이가 다 있어서 자기이해 도구로 좋습니다.

영어 페이지도 있어서 외국인 친구 공유에도 OK.
```

## currency-map (경기지역화폐) — 오픈채팅 5종

**검색 키워드**: "지역화폐", "경기페이", "성남맘", "수원맘", "31개 시·군"

### 메시지 #1 — 일반 정보 공유
```
경기지역화폐(경기페이) 받아놓고 어디서 쓸지 막막했는데, 31개 시·군 가맹점 통합 지도가 있더라고요.

https://gyeonggi-currency-map.web.app/

업종 필터 + "영업중" 토글 + 카카오톡 공유까지 됩니다. 광고 없어요.

매월 충전 인센티브 5% 그냥 날리지 마시고 이 지도로 가맹점 찾아 쓰세요.
```

### 메시지 #2 — 학원비 절감 톤
```
자녀 학원비 결제 카드로 하시는 분, 카드 포인트 0.5~1% 정도만 받지 마세요.

지역화폐로 결제하면 충전 시 6~10% 인센티브를 그대로 학원비에 쓰는 셈이에요.

가맹점 학원 확인은 https://gyeonggi-currency-map.web.app/ 사이트에서 동네 + 학원 필터.

월 30만원 학원비 → 연간 인센티브 36만원 누적 가능.
```

### 메시지 #3 — 출장·여행 톤
```
경기도 출장·여행 자주 가시는 분께.

수원·성남·용인·고양·부천 어디 가시든 한 사이트에서 가맹점 검색 가능해요.
https://gyeonggi-currency-map.web.app/

식당·카페 영업중 필터 켜고 가까운 곳 찾으면 헛걸음 X.
```

### 메시지 #4 — 자영업자 톤
```
자영업자분 우리 가게가 이 지도에 떠 있나요?
https://gyeonggi-currency-map.web.app/

가맹 등록만 되어 있으면 31개 시·군 통합 지도에 자동 노출. 광고비 없이 지역화폐 소비자가 우리 가게를 찾을 수 있는 셈이에요.

가맹 신청은 본인 거주지 시청 사회적경제과 또는 일자리경제과로 문의.
```

### 메시지 #5 — 카카오톡 공유 톤
```
가맹점 친구한테 공유할 때 일일이 주소 복사하기 번거롭잖아요.

https://gyeonggi-currency-map.web.app/ 사이트는 가맹점 핀 클릭하면 "카카오톡 공유" 버튼 있어서 단톡방에 가맹점 카드 미리보기로 1초 만에 공유 가능.

가족 단톡방에서 외식 식당 정하는 데 유용.
```

## geoinfomatic (생활권 분석) — 오픈채팅 5종

**검색 키워드**: "이사", "신혼", "청약", "학군", "임장", "부동산"

### 메시지 #1 — 임장 톤 (부동산 채팅방)
```
임장 다니실 때 동네 점수 매기는 무료 도구 있어서 공유드려요.

https://geoinfomatic.pythonanywhere.com/

매물·광고 없이 도보 30분 내 학교·병원·마트·공원 도달 시설을 분석해서 100점 점수 매겨줘요. 호갱노노에 없는 환승 0/1/2회 분리 분석도 가능.

일 5회 무료라 동네 5곳 사전 점검에 충분합니다. 임장 시즌이면 월 9,900원 Pro 결제로 PDF 저장도 가능.
```

### 메시지 #2 — 신혼 이사 톤
```
신혼 이사 검토 중이라면 동네 점수 객관적으로 매겨주는 도구 추천드려요.

https://geoinfomatic.pythonanywhere.com/

두 사람 직장 주소 입력 + 도보 30분 모드 = 동네별 100점 종합 점수.

저희도 작년에 6곳 비교해서 결정했는데 1년 살아본 결과 점수가 정확했어요.
```

### 메시지 #3 — 청약 후 거주성 톤
```
청약 당첨하셨는데 거주성 사전 확인 가능합니다.

https://geoinfomatic.pythonanywhere.com/ 사이트에 단지 주소 입력 + 도보 30분 모드.

신축은 입주 직전이라 인프라 부족 위험이 있는데, 도보 30분 시설 수가 80점 이상이면 안심 입주.
```

### 메시지 #4 — 학군지 톤 (학군 채팅방)
```
학군지 비교 시 학교·도서관·공원 도보 거리를 자동 분석해주는 도구가 있어요.

https://geoinfomatic.pythonanywhere.com/

대치 92 / 목동 88 / 평촌 86 / 분당 85 / 일산 백석 82 — 객관 점수 비교 가능.

페르소나(학원형 / 균형형 / 라이프스타일형) 따라 추천 학군지 다릅니다.
```

### 메시지 #5 — 강남 직주근접 톤
```
강남 출근 + 환승 0회 가능한 동네 분석 도구가 있어서 공유.

https://geoinfomatic.pythonanywhere.com/ → 대중교통 모드 + 환승 0회 탭

신분당선 라인(정자·미금·판교) / 9호선 / 2호선 직통 동네가 자동 표시되고, 거기서 도보 30분 인프라 점수까지 분석됩니다.
```

## 운영 흐름 (모든 사이트 공통)

### 30일 활동 → 도구 소개 흐름

| 일차 | 활동 | 목적 |
|---|---|---|
| 1~7일 | 자기소개 + 기본 질문 답 3~5개 | 신뢰 쌓기 |
| 8~14일 | 정보 공유 (도구 외 일반 정보) | 가치 제공 인식 |
| 15~21일 | 적극 답변 + 채팅방 활성화 | 친밀감 |
| 22~30일 | 자연스러운 도구 언급 1회 | 광고가 아닌 공유로 인식 |

### 멤버 닉네임 가이드

❌ 광고 의심 닉네임:
- "도구공유봇", "정보공유러", "마케팅매니저"

✅ 권장 닉네임:
- "임장중인 직장인", "신혼 이사 검토 중", "분당 거주 워킹맘", "수원 자영업자"

페르소나 명확히 = 광고 의심 안 받음.

### 운영자 사전 협의

큰 오픈채팅(500명+)은 운영자가 광고 단속하는 경우가 있으니, **사전에 운영자에게 DM**해서 도구 소개 OK인지 확인 권장.

## 절대 금지

❌ 입장 직후 도구 링크 공유 (즉시 강퇴)
❌ 단순 광고 메시지 ("저희 사이트 확인해 보세요")
❌ 여러 채팅방에 같은 메시지 일괄 공유 (스팸 봇 의심)
❌ 닉네임에 사이트명·도구명 포함
