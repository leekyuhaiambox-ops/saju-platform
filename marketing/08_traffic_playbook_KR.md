# 🚀 트래픽 유입 실전 플레이북 (한국 커뮤니티 시딩)

> **냉정한 사실**: 신생 사이트는 SEO만으로 3~6개월간 트래픽 0에 가깝다.
> 광고/제휴 인프라가 아무리 완벽해도 **방문자가 없으면 수익은 수학적으로 0**이다.
> 첫 1,000명을 데려오는 검증된 길은 딱 3가지: ① 커뮤니티 시딩 ② 바이럴 ③ 유료광고.
> 이 중 **돈 안 드는 ①②**를 이 문서가 다룬다. 자동화하면 100% 밴이라 직접 해야 한다.

---

## 0. 핵심 원칙 (이거 어기면 다 망함)

1. **"내 사이트 써주세요" = 즉시 밴/어그로.** 절대 직접 홍보 X.
2. **경험담으로 위장한다.** "이거 해봤는데 소름 돋아서 공유함" 톤.
3. **링크는 본문 말고 댓글에.** 또는 누가 "어디서 봤어?" 물으면 그때.
4. **결과 스크린샷이 핵심.** 텍스트만으론 안 퍼진다. 본인 사주 결과 캡처.
5. **계정 워밍업 먼저.** 신규 계정으로 바로 글 쓰면 스팸 처리. 며칠 댓글 활동 후.
6. **하루에 한 커뮤니티만.** 같은 글 여러 곳 동시 도배 = 밴.

## 1. 플랫폼별 전략 (효과 순)

### 🥇 1순위: 인스타그램 + 스레드 (Threads)

**왜**: 사주/운세/테스트 콘텐츠가 가장 잘 퍼지는 곳. 해시태그로 발견됨. 밴 위험 낮음.

**방법**:
- 본인 사주 결과 스크린샷 (또는 사이트의 60갑자 한자 카드 이미지) 게시
- 캡션은 `marketing/01_instagram_captions.md`의 30개 활용
- 스토리에 "내 일주 보기" 링크 스티커 (팔로워 무관 누구나 클릭)
- **릴스가 핵심**: `marketing/07_youtube_shorts_scripts.md`의 대본으로 15~30초 릴스

**복붙 게시 예시**:
```
나 사주 일주가 "경진(庚辰)"인데
"큰 흙더미 속의 광석, 잠재력 큰 인재" 래 ㅋㅋㅋ
야망 강하고 추진력 있다는데 인정...😳

무료로 보는 곳 프로필 링크에 있음
너네 일주도 봐바 (댓글로 알려줘)

#사주 #사주풀이 #무료사주 #일주 #오늘의운세 #MBTI #성격테스트 #운세
```

---

### 🥈 2순위: 더쿠 / 인스티즈 (여성 위주 커뮤니티)

**왜**: 사주·타로·MBTI·테스트 콘텐츠에 가장 반응 좋은 층. 단, 홍보 규정 엄격.

**규칙 주의**:
- 더쿠: 외부링크/홍보 글 삭제됨. **"유머/일상" 게시판에 경험담**으로.
- 인스티즈: "익명잡담/일상" 게시판. 마찬가지로 경험담 위장.
- 링크는 본문 X → 댓글에 "궁금하면 ㅇㅇ" 식으로 또는 누가 물으면.

**복붙 게시 예시 (더쿠 일상 게시판)**:
```
제목: 나 사주 일주 봤는데 너무 정확해서 소름 돋음 ㅋㅋㅋ

심심해서 무료 사주 보는 거 했는데
내 일주(태어난 날) 설명이 진짜 나랑 똑같아서 놀람...
"한번 결심하면 끝까지 밀어붙인다"는데 ㅇㅈ
띠로 보는 거 말고 태어난 날로 보는 거라 더 정확한듯?

너희도 일주별로 성격 다른데 뭔지 알아? 댓글로 ㄱㄱ
```
→ 댓글 달리면 "○○에서 봤어 무료임" 자연스럽게.

---

### 🥉 3순위: 네이트판 / 에펨코리아 / 디시

**네이트판**: 사연·테스트 인기. "오늘의 운세/사주" 톤으로 자유게시판.
**에펨코리아 (유머)**: "ㅋㅋ" 톤 필수. 진지하면 묻힘.
**디시 (점術갤, 해당 갤러리)**: 점·사주 갤러리는 오히려 환영. 다른 갤은 어그로 위험.

**디시 점술갤 복붙 예시**:
```
제목: 24절기 + 야자시 보정 되는 무료 만세력 사이트 찾음

대부분 무료 사이트가 입춘을 2월 4일 0시로 퉁치는데
이건 메우스 천문공식으로 절기를 분 단위로 계산하더라.
야자시(23시 이후) 일주 보정도 되고.
60갑자 일주 풀이도 다 있음. 광고만 있고 무료임.

[링크]
```
→ 점술갤은 정확도 따지는 사람 많아서 "정확함" 어필이 통함.

---

### 4순위: 카카오톡 오픈채팅 / 단톡방

**왜**: 한국 바이럴의 진짜 엔진. 친구·지인 단톡방 1개 공유 = 5~10명 유입.

**방법**:
- 본인 결과 페이지 링크를 단톡방에 공유 (OG 카드 미리보기 뜸)
- "나 사주 봤는데 ㅋㅋ 너네도 해봐" 한 줄
- 사주/운세 관련 오픈채팅방 검색해서 가입 후 자연스럽게

---

### 5순위: 클리앙 / 뽐뿌 / 보배드림 (남성·일반)

**클리앙 "모두의공원"**, **뽐뿌 "자유게시판"** — 기술·일상 잡담.
개발 스토리로 접근하면 호응: "주말에 사주 계산기 만들어봤습니다" (개발자 커뮤니티 호응).

**클리앙 복붙 예시**:
```
제목: 토요일에 사주(만세력) 계산기 직접 만들어봤습니다

24절기를 천문 알고리즘으로 분 단위 계산하고
야자시 보정까지 넣었더니 기존 무료 사이트보다 정확하네요.
Flask + 순수 파이썬으로 짰고 무료로 열어뒀습니다.
혹시 본인 사주 정확한지 확인해보실 분 (피드백 환영)

[링크]
```
→ 개발 과정 어필 = 홍보 아닌 "공유"로 받아들여짐.

## 2. 첫 2주 실행 시퀀스 (순서대로)

| Day | 할 일 | 소요 |
|---|---|---|
| 1 | 인스타 계정 만들고 본인 사주 결과 + 60갑자 카드 3개 게시 | 20분 |
| 2 | 스레드(Threads) 같은 콘텐츠 게시 + 해시태그 | 10분 |
| 3 | 단톡방·지인에게 본인 결과 링크 공유 ("나 이거 해봤어") | 5분 |
| 4 | 더쿠 일상게시판 경험담 글 1개 | 15분 |
| 5 | 인스티즈 익명잡담 경험담 글 1개 | 15분 |
| 6 | 디시 점술갤 "정확한 만세력" 글 | 10분 |
| 7 | 인스타 릴스 1개 (쇼츠 대본 활용) | 30분 |
| 8 | 클리앙 모두의공원 "개발 스토리" 글 | 15분 |
| 9 | 네이트판 자유게시판 경험담 | 15분 |
| 10 | 카카오톡 사주 오픈채팅방 2개 가입 + 자연스럽게 | 10분 |
| 11~14 | 반응 좋았던 채널 위주로 콘텐츠 2~3개 추가 | 매일 15분 |

**하루 15분씩, 2주.** 이게 첫 트래픽의 씨앗이다.

## 3. 절대 하지 말 것 (밴 지름길)

- ❌ 같은 글을 5개 커뮤니티에 동시 복붙 도배
- ❌ 신규 계정으로 가입 당일 홍보 글
- ❌ 본문 첫 줄에 링크 박기
- ❌ "무료니까 와주세요" 식 노골적 구걸
- ❌ 댓글에 같은 링크 반복 도배
- ❌ 봇으로 자동 게시 (인스타·더쿠·디시 전부 즉시 밴)

## 5. 바이럴 증폭 장치 (사이트에 이미 심어둠)

1. **결과 페이지 궁합 배너** — "좋아하는 사람과 궁합 보기" (본인 정보 자동 입력 → 친구 데려옴)
2. **동적 OG 카드** — 결과 공유 시 본인 사주가 그려진 이미지 미리보기
3. **카카오톡 공유 버튼** — 결과 페이지 상단
4. **8개 채널 공유 버튼** — 텔레그램·왓츠앱·라인·X·레딧 등

→ 사용자가 위 커뮤니티에서 1명 데려오면, 그 1명이 궁합/공유로 2~3명 더 데려오는 구조.

## 6. 한 달 후 판단

UTM 데이터로 ROI 좋은 채널 1~2개 확정 → 거기 집중.
트래픽 일 100명 넘으면 → 광고 수익 발생 시작 + 유료 광고 검토 가치 생김.

**기억할 것**: 사주 사이트는 시즌(연말·신정·입춘) 트래픽이 폭발한다.
지금 씨앗 뿌려두면 연말에 SEO + 입소문이 누적돼 터진다.
