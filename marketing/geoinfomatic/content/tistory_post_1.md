## 들어가며

한국의 부동산 도구 시장은 매물·실거래가 중심이다. 호갱노노·직방·다방·KB부동산은 모두 매물을 메인 콘텐츠로 노출하며, 생활권 분석은 부가 기능에 머무른다. **그러나 이사·청약·임장 의사결정의 핵심은 매물보다 동네의 생활 인프라**다.

본 글에서는 매물·광고 없이 순수 생활권 접근성만 분석하는 무료 도구 **생활권 접근성 분석**(https://geoinfomatic.pythonanywhere.com/)을 리뷰한다.

## 핵심 결론

| 항목 | 평가 |
|---|---|
| 차별점 | 매물 0개 노출 — 순수 분석 도구 |
| 분석 정밀도 | 도보 + 대중교통, 환승 0/1/2회 분리 |
| 시설 종류 | 8종 (학교·병원·마트·공원·헬스장·약국·도서관·카페) |
| 가격 모델 | 일 5회 무료 + 월 9,900원 Pro |
| 적합 사용자 | 이사 검토자, 임장 다니는 투자자, 학군 비교 부모 |

## 기능 상세 리뷰

### 1. 아이소크론(Isochrone) 분석

도구의 핵심은 **아이소크론** — 특정 위치에서 도보 또는 대중교통으로 N분 안에 도달 가능한 지역의 경계를 그리는 GIS 기법이다.

사용자가 위치(주소) + 모드(도보/대중교통) + 시간(10/20/30/45분)을 입력하면, 지도 위에 **도달 가능 구역이 색칠된 폴리곤**으로 표시된다. 그 안에 있는 시설들이 자동으로 분석 대상이 된다.

### 2. 다중 시설 동시 분석

분석 가능한 시설은 다음 8종:

| 시설 | 활용 예 |
|---|---|
| 학교 | 자녀 학군 분석 |
| 병원 | 응급실 도달, 시니어 거주 |
| 대형마트 | 생활 편의 |
| 공원 | 운동·여가 |
| 헬스장 | 운동 인프라 |
| 약국 | 의료 접근성 |
| 도서관 | 학습 환경 |
| 카페 | 직주 + 라이프스타일 |

각 시설은 페르소나에 따라 우선순위가 달라지는데, **온보딩 단계에서 본인이 중요시하는 시설을 선택**하면 점수에 가중치가 반영된다.

### 3. 환승 횟수 분리

대중교통 모드의 차별점은 **환승 횟수별 결과 분리**다:

- 환승 0회: 직통 도달 가능 영역
- 환승 1회: 추가 1회 환승 시 확장 영역
- 환승 2회: 2회 환승 시 추가 영역
- 환승 3회+: 그 외

기존 호갱노노·직방의 "지하철 도보 5분"이라는 단순 지표 대비, 실제 출퇴근 동선의 부담을 훨씬 정밀하게 보여준다.

### 4. 종합 점수 + AI 요약

분석 완료 시 **100점 만점 종합 점수**와 자연어 AI 요약이 생성된다. 예시:

> 분당구 정자동 = 87점 (A+ 등급)  
> "직주 근접 우수 (강남·판교 환승 0회), 학군 강점 (반경 30분 학교 12곳), 마트 접근성 평균 미만 (대형 마트 1곳, 도보 28분)"

이 요약 형식은 **의사결정을 가속**하는 핵심. 점수만 봐선 알기 어려운 강·약점을 한 문장으로 정리해 준다.

### 5. 역별 커뮤니티 댓글

지하철역 핀 클릭 시 표시되는 모달에 **역별 커뮤니티 댓글 섹션**이 있다. 호갱노노에도 단지별 댓글이 있지만, 본 도구는 **역 단위**라 더 일반적인 동네 정보 공유가 가능하다.

## 기술적 관찰

| 항목 | 값 |
|---|---|
| Backend | Python Flask, PythonAnywhere |
| Frontend | Vanilla JS + Leaflet 1.9.4 |
| 지도 데이터 | OpenStreetMap + 한국 공공 데이터 |
| 도보 라우팅 | OSRM (Open Source Routing Machine) 추정 |
| 대중교통 그래프 | 자체 구축 (지하철·버스 노선 데이터) |
| AI 요약 | 외부 LLM API 추정 |
| robots.txt | 정교 — Googlebot·NaverBot 허용, AI 학습 봇 차단 |
| Sitemap | ⚠️ 메인 URL 누락 (개선 여지) |

robots.txt에서 GPTBot·CCBot·ClaudeBot·PerplexityBot·Google-Extended 등 AI 학습 크롤러를 명시적으로 차단한 점이 인상적. 사용자 데이터 보호를 위한 의도적 선택으로 보인다.

## 페르소나별 사용 시나리오

### A. 신혼 이사 검토자

직주 근접 + 마트·공원·카페 = 라이프스타일 점수. 도보 30분 모드로 신축 아파트 일대를 돌려보면 동네 인프라 한눈에 파악 가능.

### B. 학군 비교 부모

학교·도서관·학원가(카테고리에 학원은 없지만 학교로 어림) 우선순위로 가중치. 도보 20분 모드로 자녀가 직접 다닐 수 있는 시설 확인.

### C. 부동산 임장 투자자

후보 매물 5~10곳을 하루에 돌리고, 동네 점수로 정렬. Pro의 **멀티 포인트 비교** 기능이 임장 정리에 효율적.

### D. 청약 당첨 후 거주성 확인

신축 단지 주변 30분 분석 → 실제 살만한 동네인지 객관적 판단.

## 호갱노노 vs 본 도구 — 정밀 비교

| 기능 | 호갱노노 | 생활권 접근성 분석 |
|---|---|---|
| 매물·시세 | ✓ (메인) | ✗ (의도적) |
| 실거래가 | ✓ | ✗ |
| 생활권 점수 | ✓ (단순) | ✓ (정밀) |
| 도보 아이소크론 | ✗ | ✓ (10~45분) |
| 환승 분석 | ✗ | ✓ (0/1/2회) |
| 다중 시설 동시 분석 | △ | ✓ (8종) |
| AI 자연어 요약 | ✗ | ✓ |
| PDF 리포트 | ✗ | ✓ (Pro) |
| 멀티 포인트 비교 | ✗ | ✓ (Pro) |
| 가격 | 무료 (광고+매물) | 무료 5회 / 월 9,900원 |

**상호 보완** 관계다. 호갱노노로 시세·매물 확인 → 본 도구로 생활권 객관 분석. 임장 자체는 카카오맵으로 실제 거리 재확인.

## 개선 여지 (관찰 사항)

1. **Sitemap.xml에 메인 URL `/` 누락** — `/profile/*` URL 28개만 등록되어 있어 검색엔진이 메인 서비스를 색인하지 못함. 이는 검색 트래픽 부재의 직접 원인일 가능성.
2. **Pro 결제 페이지 미구현** — Pro 모달은 있지만 결제는 "준비 중" alert 상태. Toss Payments 연동 시급.
3. **모바일 카카오톡 공유 SDK 없음** — 분석 결과 공유 시 단순 URL 복사만 가능.

## 결론

**한국 부동산 도구 시장에서 보기 드문 "매물 없는 순수 분석 도구"**. 호갱노노·직방이 매물 중심 UX로 분석을 부가 기능에 두는 것과 대비된다.

기술적으로는 PythonAnywhere 무료 티어 위에서 정밀한 isochrone 분석을 구현한 점이 돋보인다. 외산 GIS 도구의 비용을 부담하기 어려운 한국 인디 개발자가 어떻게 풀스택 분석 도구를 만들 수 있는지 보여주는 사례.

이사·임장·청약 검토하는 사용자라면 시도해 볼 가치가 충분하다. 일 5회 무료라서 부담 없고, 본격 임장 시즌엔 1만 원 Pro 결제만으로 PDF 리포트까지 받을 수 있다.
