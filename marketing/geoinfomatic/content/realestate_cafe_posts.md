# 부동산 카페 게시 카피 — 생활권 접근성 분석

> **운영 원칙**:
> - 부동산스터디(다음카페) / 호갱노노 자유게시판 / 부동산갤러리 등에 분산
> - 1주 1게시, 같은 카피 X
> - 본문에서 부동산 인사이트 제공 후 도구는 도구로 소개
> - 광고 의심 받지 않게 "발견" / "사용기" / "비교" 톤

---

## 카피 풀 (4가지 톤)

### 톤 A — 임장 노하우 공유형 (부동산스터디·호갱노노 자유게시판)

**제목**: 임장 다닐 때 동네 점수 매기는 무료 도구 찾았는데 비교해 봤습니다

```
부동산 임장 다니면서 같은 단지 안에서도 동네마다 점수 차이가 큰 거 경험하시잖아요.

도보 10분에 마트 있는지, 지하철까지 정확히 몇 분 걸리는지, 학교·병원이 어디까지 도달 가능한지. 저는 매번 카카오맵에서 일일이 찍어보면서 노트에 정리하는 게 너무 비효율적이었습니다.

호갱노노 생활권 점수도 있는데 너무 단순하고(학군·교통·생활편의 3개 카테고리), 직방·다방은 그냥 매물만 보여주죠.

최근에 발견한 도구가 진짜 잘 만들어져 있어서 공유합니다.

생활권 접근성 분석: https://geoinfomatic.pythonanywhere.com/

특징:
- 도보 또는 대중교통 모드 분리 (저는 차 있어서 대중교통만 봤어요)
- 10/20/30/45분 슬라이더로 도달 범위 시각화 (아이소크론 폴리곤)
- 학교·병원·마트·공원·헬스장·약국·도서관·카페 동시 분석
- 환승 0회 / 1회 / 2회 분리 결과 (이게 호갱노노에 없는 기능)
- 100점 만점 점수 + AI 한 문장 요약

매물·광고 없이 순수 분석만 해서 임장 사전 점검용으로 좋습니다.

일 5회는 무료고, 그 이상은 월 9,900원 Pro인데 임장 시즌에만 결제해도 됩니다.
PDF 리포트 저장이 가능해서 임장 노트 만들기 진짜 편해요.

저는 임장 한 달 동안 동네 23곳 돌려보고 비교했는데, 1만 원으로 가성비 짱이었습니다.

광고 아니에요. 그냥 부동산 도구 공유 :)
```

### 톤 B — 신혼 이사 정보형 (네이버 신혼 카페)

**제목**: 신혼 이사 동네 비교할 때 진짜 유용했던 도구 공유

```
저희도 작년에 신혼 이사 동네 6곳 비교하면서 정말 머리 아팠어요.

호갱노노 켜고 시세 보고, 카카오맵 길찾기 일일이 찍고, 학구도 사이트 돌고, 직방 매물 검색하고… 이걸 동네마다 반복하니까 결정을 못 하고 한 달 흘러갔습니다.

남편이 강남 출근이라 환승 0회로 갈 수 있는 동네를 우선 봤는데, 호갱노노엔 그런 분석 기능이 없더라고요.

그러다 발견한 도구가 생활권 접근성 분석(https://geoinfomatic.pythonanywhere.com/) 사이트인데, 정말 신혼 이사용으로 딱이에요.

- 두 사람 직장 주소 둘 다 입력 가능
- 대중교통 환승 0회 / 1회 / 2회 분리 결과
- 도보 30분 내 마트·카페·공원 자동 점수
- 100점 점수로 동네 6곳 한 번에 비교

저희는 결국 신분당선 라인(미금-정자-수내) 우선 검토해서 정자동으로 결정했어요. 1년 살아본 결과 점수가 맞았습니다.

매물은 호갱노노에서 따로 보시고, 동네 점수만 객관적으로 보고 싶으면 이 도구 추천합니다.

일 5회 무료라 신혼 이사 동네 검토용으론 충분해요.
```

### 톤 C — 학군지 정보형 (학군 카페)

**제목**: 학군지 도보 거리 분석 — 학교·도서관 한 번에 비교하는 도구

```
자녀 학군 보고 이사 결정할 때, 학교·도서관·학원·공원 도보 거리를 일일이 찍어보는 거 너무 번거롭잖아요.

대치동·목동·평촌·분당 정자동·일산 백석동… 후보 동네가 많은데 동네별로 도보 30분 내 학교가 몇 개인지, 도서관까지 몇 분인지 객관적 비교 도구가 없었어요.

저는 [생활권 접근성 분석](https://geoinfomatic.pythonanywhere.com/) 사이트로 학군 점수 매기고 있는데 진짜 편합니다.

- 초기 설정에서 "학교·도서관·공원" 우선순위 켜면 학군 점수 가중치 반영
- 도보 30분 모드로 자녀가 직접 다닐 수 있는 시설만 표시
- 100점 점수 + AI 요약으로 강점·약점 한 문장

대치동 90점+ / 목동 88점 / 평촌 86점 / 분당 정자 85점 / 일산 백석 82점 정도 나왔어요. 이걸 기준으로 시세 비교까지 하면 객관적 의사결정 가능합니다.

매물은 호갱노노·KB부동산 따로 보시고요. 이 도구는 학군 점수에만 집중한 도구라서 객관적이에요.

일 5회 무료, Pro 월 9,900원이라 학군 검토 시즌에만 결제해도 OK.

자녀 학군 검토하시는 분께 공유드립니다.
```

### 톤 D — 청약 거주성 점검형 (청약 카페)

**제목**: 청약 당첨 단지 거주성 점검 — 신축 인프라 부족 미리 확인하는 법

```
청약 당첨됐는데 정작 입주하니까 주변 인프라가 부족해서 후회하시는 분 많죠.

신축 단지는 입주 직전이라 도보 시간을 카카오맵에서 정확히 찍기 어려운 게 함정이에요. 단지는 멋있는데 도보 30분에 마트 1개·학교 1개뿐이면 거주성 떨어집니다.

저는 청약 당첨 후 [생활권 접근성 분석](https://geoinfomatic.pythonanywhere.com/) 사이트로 사전 점검합니다.

- 단지 주소 입력 → 도보 30분 모드
- 학교·병원·마트·공원·헬스장·약국·도서관·카페 도달 시설 수 분석
- 100점 점수 + AI 요약

점검 기준:
- 80점+ : 인프라 안정, 안심 입주
- 70~80점 : 평균, 5년 후 보강 기대
- 70점 미만 : 인프라 부족, 입주 후 불편 예상

호갱노노에선 단지 자체 정보만 보여주는데, 이 도구는 단지 주변 30분 인프라까지 객관 점수로 보여줘서 청약 후 의사결정에 정말 유용해요.

일 5회 무료라 부담 없이 후보 단지 5곳 정도 비교 가능합니다.

청약 검토하시는 분께 도움 되셨으면 좋겠어요.
```

## 대상 카페·게시판 매트릭스

| 카페·게시판 | 회원 수 | 톤 매칭 | 게시 빈도 권장 |
|---|---|---|---|
| 부동산스터디 (다음카페) | 200만+ | A (임장) | 월 1회 |
| 호갱노노 자유게시판 | (호갱노노 앱 내) | A (임장) | 월 1회 |
| 디씨 부동산 갤러리 | 활발 | A 또는 D | 분기 1회 |
| 네이버 신혼부부 카페 | 다수 | B (신혼) | 카페별 월 1회 |
| 네이버 학군 카페 (서울·분당·평촌) | 동네별 | C (학군) | 동네별 분기 1회 |
| 청약홈 정보 카페 (네이버) | 다수 | D (청약) | 월 1회 |
| 더쿠 — 부동산 채널 | 활발 | A 또는 B | 분기 1회 |
| 클리앙 새소식 | 활발 | A | 분기 1회 (스팸 주의) |

## 게시 후 알고리즘 활용

1. **댓글 토론 유도** — 마지막에 "여러분은 어떤 도구 쓰세요?" 같은 질문으로 댓글 활성화
2. **24시간 답글** — 모든 댓글에 24시간 내 답글
3. **시리즈화** — 1주 후 같은 카페에 "그 도구 사용 결과 공유" 후속 글

## 절대 금지 사항

❌ 같은 카피 여러 카페 동시 게시 → 차단·신고
❌ "광고", "마케팅", "구독 권유" 명시
❌ Pro 요금 단순 강조 — "임장 시즌 결제 권장" 같은 가치 메시지로 변환
❌ 카페 운영자가 광고 금지 명시한 카페에서 게시
